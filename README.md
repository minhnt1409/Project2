## API
<hr>

### 1. Signup
- **Link**: 
    ```
        http://localhost:3000/user/signup
    ```
- **Type**: POST
- **Request**:
  
    ```
        {   
            "username": "user3",
            "password": "3"
        }
    ```
- **Respone**:
  
    ```
        {
            "code": "1000",
            "message": "OK",
            "data": "null"
        }
    ```
